# Installation Notes:
# ===================

# 1. Gunakan virtual environment untuk isolasi dependencies:
# python -m venv alice_env
# source alice_env/bin/activate  # Linux/macOS
# alice_env\Scripts\activate     # Windows

# 2. Upgrade pip sebelum instalasi:
# python -m pip install --upgrade pip

# 3. Install requirements:
# pip install -r requirements.txt

# 4. Untuk development environment:
# pip install -r requirements.txt[dev]

# 5. Untuk production environment (minimal dependencies):
# pip install -r requirements-prod.txt
